# BaseQuest Rewards Project Rules

## Project Goal

Build a production-ready Base Mini App called BaseQuest Rewards.

The app targets the Base ecosystem first and Farcaster second.

## Tech Stack

- Next.js

- TypeScript

- Tailwind CSS

- OnchainKit

- Base Mini App SDK

- Farcaster Mini App SDK

- wagmi

- viem

## Development Rules

- Never rewrite the entire project.

- Modify only the requested files.

- Preserve existing functionality.

- Keep components reusable.

- Keep the UI mobile-first.

- Use clean architecture.

- Use TypeScript best practices.

- Ask before installing new dependencies.

- Never remove working code unless requested.

## UI Style

- Base Blue (#0052FF)

- Glassmorphism

- Rounded cards

- Smooth animations

- Modern Web3 design

## MVP Features

- Wallet Connection

- Wallet Information

- ETH Balance

- Daily Check-in

- Rewards

- Streak Counter

- Profile

- Settings

## Important

Always explain what files will be modified before generating code.
